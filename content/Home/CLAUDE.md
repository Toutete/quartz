이것은 지식 vault 및 코드 저장소입니다. 사용자가 코드 작성 및 개념 정리를 요청할 때 다음 규칙을 엄격히 따르세요.

1. 구조적 분리:
   - 작성된 코드(.py, .m 등)는 `projects/` 폴더 내 적절한 하위 폴더에 저장합니다.
   - 코드의 수학적/물리적 배경 이론과 설명은 `concepts/` 폴더에 마크다운(.md)으로 저장합니다.
   - 마크다운 노트 하단에 해당 코드 파일의 상대 경로 링크를 추가하세요.

2. 수식 표현 (LaTeX):
   - 수학 공식 및 방정식은 반드시 LaTeX 문법을 사용합니다.
   - 인라인 수식은 $E = mc^2$ 처럼 단일 달러 기호를 사용합니다.
   - 블록 수식은 이중 달러 기호($$)를 사용하여 중앙 정렬되게 작성합니다.

3. Frontmatter 필수 포함:
   - 웹에 공개할 이론 노트에는 `is_public: true`를 추가합니다.

4. 자동 배포 (Git):
   - 파일 생성 및 수정이 완료되면 반드시 `git add .`, `git commit -m "[변경 요약]"`, `git push origin main` 명령어를 실행하여 변경 사항을 GitHub에 반영하세요.
